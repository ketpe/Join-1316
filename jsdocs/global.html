(constant) firebaseConfig :Object

Firebase configuration object for the Join-1316 project. This configuration is used to initialize the Firebase app and connect to the Firebase services.
Type:
  • Object
Properties:
Name Type Description
apiKey string The API key for the Firebase project.
authDomain string The authentication domain for the Firebase project.
databaseURL string The URL of the Firebase Realtime Database.
projectId string The project ID for the Firebase project.
storageBucket string The storage bucket for the Firebase project.
messagingSenderId string The messaging sender ID for the Firebase project.
appId string The application ID for the Firebase project.
