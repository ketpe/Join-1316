Namespace: navigations

navigations

This script contains functions for navigating between different pages of the application. It includes functions to navigate to the sign-up page, summary page, contact page, privacy policy, legal notice page, and help page. It also includes functions to navigate to the add task page and board page, as well as setting the active state of navigation buttons based on the current view.
