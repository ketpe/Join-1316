Class: AddTaskUtils

AddTaskUtils()

Utility class for managing tasks, contacts, and subtasks. Provides methods for handling user IDs, contact assignments, and subtask management. Also includes caching functionality for task data in the add task view. Dependencies: - db-functions.js: For database operations. - addTaskCreateTask.js: For task creation processes. Usage: const addTaskUtils = new AddTaskUtils();
