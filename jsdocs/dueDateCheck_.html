dueDateCheck

Class for validating and managing due date input for tasks. Ensures the due date is in the correct format (DD/MM/YYYY) and not in the past. Provides feedback to the user through UI components.

(static) checkIsCorrectDate(valueToCheck) → {boolean}

Check whether the date value conforms to the format. Split the string into its individual date components. Create a date from these numbers; this date cannot be in the past. Finally, the date components from the created date are compared with the components of the passed date. If they are equal, it returns true; otherwise, it returns false.
