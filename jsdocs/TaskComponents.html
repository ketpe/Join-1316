Class: TaskComponents

TaskComponents(currentUser, currentInstance)

Class to manage task components including contacts, categories, priorities, and subtasks. This class provides methods to load and manipulate task-related data. It interacts with Firebase for data retrieval and updates the UI accordingly. It also includes validation for task title and due date. It is designed to be used in a task management application. Example usage: const taskComponents = new TaskComponents(currentUser, 'taskInstance');
